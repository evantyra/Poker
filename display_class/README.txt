This is the Display class for the ECE373 project. It includes the Display
class and example Makefile and main.cpp files. You are allowed to modify
the code in your project.

########################
# To run the example:
########################

1. Set your putty configuration to UTF-8
	"Change Settings"->"Window"->"Translation"-> 
	"Remote character set:"->"UTF-8"
2. Do not run a screen session while running the program.
3. Type "make"
4. Type "./display"
